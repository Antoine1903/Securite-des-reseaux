# Administration d’infrastructures Linux

## Objectif
Projet d’initiation à l’administration de services d’infrastructure sous Linux, incluant annuaire, base de données, reverse proxy et stockage clé-valeur.

## Services mis en œuvre
- OpenLDAP : déploiement et gestion d’un annuaire
- PostgreSQL : gestion des rôles et permissions
- Nginx : configuration en reverse proxy et load balancing
- Redis : stockage clé-valeur et persistance

## Compétences travaillées
- Administration système Linux
- Gestion des services réseau
- Contrôle des accès et permissions
- Architecture d’infrastructure
- Notions de haute disponibilité (load balancing)

## Remarque
Projet réalisé dans un contexte académique d’apprentissage des bases de l’administration système et réseau.
